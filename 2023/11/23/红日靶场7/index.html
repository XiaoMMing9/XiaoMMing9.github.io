192.168.1.4

goby扫描

redis写ssh公钥并进行ssh连接

将192.168.1.4连上Cs控

下方插件可以将linux机器加入到Cs控中

生成木马t_cc2.out

使用以下命令将t_cc2.out传到192.168.1.4的root目录下

192.168.1.4运行t_cc2.out就上线了

收集192.168.1.4的内网信息

双网卡机器，存在192.168.52.0/24网段

发现web服务代理于192.168.52.20:8000

192.168.52.20

CVE-2021-3129得shell

发现是docker环境

提权

/home/jobs/shell是一个用户自己编写的可执行文件

发现这个程序会运行system(‘ps’)

修改环境变量，使shell文件在执行ps程序的时候优先执行/var/www/html下的ps程序(/bin/bash)

Bug:哥斯拉下的shell不能修改环境变量，还是需要将shell反弹到192.168.1.4上

修改环境变量，获取root权限

docker逃逸成root权限

--privileged 是 Docker 的一个安全特权设置，允许容器在主机上执行一些敏感的系统级任务。启用 --privileged 选项的 Docker 容器将拥有对主机系统内核的完全访问权限

而该docker容器就是 依靠 特权模式 运行的

192.168.1.4这台web机器创建的ssh公钥传递给他使用

在docker特权模式下将这个文件用echo命令将公钥写入到/var/www/html/ming/root/.ssh/authorized_keys 下，

ssh root@192.168.52.20 直接登录
